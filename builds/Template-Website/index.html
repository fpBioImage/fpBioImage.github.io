Welcome

This is a template website to use FPBioimage, a new piece of software developed at the University of Cambridge. For more information, please see the project page.
Note that to run FPBioimage offline, your website will have to be running on a localhost. See the project page for more information.

We recommend Google Chrome for this site.
The tool will run on most web browsers, but 32-bit users may have memory errors on large data sets. If this happens, try downloading a 64-bit version of your browser.
